Run the file "SeeSawStars.py" to play the game.

Cheat codes for this game are:
No tip
Reach for the stars
Dr.Chesney never loses

See Saw Stars is a game that was developed as a part of the Engineering 100 curriculum at the University of Michigan. The game aims to provide a therapeutic effect to individuals diagnosed with autism spectrum disorders.

Although all contributions to this repo are from a single account (mine), development from January 2015 - April 2015 was also done by Charles Bell, Jason Ye, and Ryan Pifer.
